# Практическое задание 2: Структуры данных и алгоритмы на Go

## Описание
Решение второго практического задания. Реализованы базовые структуры данных (стек, очередь, односвязный список), конвертер римских цифр в арабские, а также алгоритм заполнения двумерного массива случайными уникальными числами.
