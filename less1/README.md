# Практическое задание 1: Сортировка массива на Go

## Описание
Решение первого практического задания. Реализована **пузырьковая сортировка** (Bubble Sort) для статического массива из 100 целых чисел.

## Как запустить
1. Убедитесь, что на компьютере установлен компилятор Go.
2. Откройте терминал в папке проекта.
3. Выполните команду:
   ```bash
   go run main.go
