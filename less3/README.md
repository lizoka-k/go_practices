# Практическое задание 3: Дженерики и пакеты в Go

## Описание
Рефакторинг второго практического задания:
- Структуры данных (стек, очередь, односвязный список) вынесены в отдельные пакеты
- Все структуры данных реализованы с использованием дженериков (generics)
- Функции преобразованы в методы структур

## Структура проекта
```text
less3/
├── stack/     - пакет стека
├── queue/     - пакет очереди
├── list/      - пакет односвязного списка
├── main.go    - демонстрация работы
└── go.mod     - модуль Go
```

## Как запустить
1. Убедитесь, что на компьютере установлен Go версии 1.18 или выше
2. Откройте терминал в папке проекта (less3)
3. Выполните команду:
   ```bash
   go run .
   ```
